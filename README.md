# Dokumentanonymiserare

Ett enkelt verktyg för att anonymisera personuppgifter i Word-dokument.

## Säkerhet

- Dokument lagras inte permanent
- Bearbetning sker endast i minnet
- Inga personuppgifter loggas
- Krypterad trafik (HTTPS) används vid webbhosting

## Användning

1. Ladda upp ett Word-dokument
2. Granska identifierade personer
3. Klicka på anonymisera
4. Ladda ner dokumentet
